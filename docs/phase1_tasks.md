# Phase 1: 内核与MessagePack - 任务清单

> **阶段**: Phase 1  
> **周期**: Week 1-4  
> **目标**: 微内核稳定，MessagePack协议定型  
> **交付**: MathCore Kernel 6.0，MessagePack序列化层，ComputeExt (Rust)

---

## 1. 阶段目标

### 1.1 核心指标

| 指标 | 目标 | 验收标准 |
|------|------|---------|
| MessagePack解析延迟 | <1ms | 基准测试 |
| 内核代码行数 | <5000行 | cloc统计 |
| 单元测试覆盖率 | >80% | cargo-tarpaulin |
| 计算功能可用性 | 基本代数/微积分 | 功能测试 |

### 1.2 交付物

1. **MathCore Kernel 6.0** - 微内核核心
2. **MessagePack序列化层** - 协议v6实现
3. **ComputeExt (Rust)** - 本地计算扩展
4. **基础CLI** - 命令行接口

---

## 2. 任务拆解

### 2.1 任务 T1: 项目骨架搭建

**角色**: Build (hephaestus)

**任务内容**:
- 创建 Cargo workspace 结构
- 配置依赖版本 (依赖表A)
- 搭建目录结构
- 配置 CI/CD 流水线

**依赖**: 无

**验收标准**:
- [ ] workspace 编译通过
- [ ] 所有依赖版本锁定
- [ ] GitHub Actions 编译通过

**预估工时**: 2小时

---

### 2.2 任务 T2: 微内核核心

**角色**: Coder (hephaestus)

**任务内容**:
- 实现 kernel-core 模块
  - 进程隔离管理
  - 资源配额控制
  - 插件生命周期
  - 热更新协调
- 实现 kernel-bus 模块
  - 消息路由
  - 负载均衡
  - 事件广播

**前置任务**: T1

**验收标准**:
- [ ] 内核代码 < 5000行
- [ ] 插件加载/卸载正常工作
- [ ] 消息路由功能测试通过

**预估工时**: 3天

---

### 2.3 任务 T3: MessagePack 协议层

**角色**: Coder (hephaestus)

**任务内容**:
- 实现 `MathMessage` 结构
- 实现 `MsgHeader` / `MsgPayload`
- 实现序列化/反序列化
- 实现协议版本协商

**前置任务**: T1

**验收标准**:
- [ ] 序列化延迟 < 1ms (1000次平均)
- [ ] 与Python客户端互操作测试通过
- [ ] 协议版本v6文档完成

**预估工时**: 2天

---

### 2.4 任务 T4: 计算扩展 ComputeExt

**角色**: Coder (hephaestus)

**任务内容**:
- 实现基本代数运算
  - 四则运算
  - 幂/对数
  - 三角函数
- 实现微积分基础
  - 求导
  - 积分(基础)
- 实现表达式解析
  - AST构建
  - 化简

**前置任务**: T2

**验收标准**:
- [ ] 基本代数测试通过
- [ ] 求导测试通过
- [ ] 表达式解析测试通过

**预估工时**: 4天

---

### 2.5 任务 T5: CLI 基础接口

**角色**: Coder (hephaestus)

**任务内容**:
- 实现命令行解析
- 实现计算命令 `mathcore compute`
- 实现配置命令 `mathcore config`
- 实现版本命令 `mathcore version`

**前置任务**: T3

**验收标准**:
- [ ] help 输出正确
- [ ] compute 命令功能正常
- [ ] 错误处理完善

**预估工时**: 1天

---

### 2.6 任务 T6: 单元测试与基准

**角色**: Reviewer (hephaestus)

**任务内容**:
- 编写核心模块单元测试
- 编写集成测试
- 搭建基准测试框架
- 性能测试报告

**前置任务**: T2, T3, T4

**验收标准**:
- [ ] 覆盖率 > 80%
- [ ] 所有测试通过
- [ ] 性能报告生成

**预估工时**: 2天
